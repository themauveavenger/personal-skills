# Personal Skills

A repository of skills I find useful. For whatever.
I probably lifted or re-worded some of these from around the internet and I'll give credit to original authors via this README as best I can.

## Layout

Skills live under `skills/`, one directory per skill, each containing a `SKILL.md`.

## Install with `npx skills`

```bash
npx skills add themauveavenger/personal-skills --list
npx skills add themauveavenger/personal-skills --skill oracle-reviewer
```
